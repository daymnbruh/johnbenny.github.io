2000W Load Control

STM32G474 Buck Converter LT3845AIFE Proteus Keil µVision
ABOUT THIS PROJECT

Overview

STM32G474-based load controller featuring an LT3845AIFE synchronous buck controller, a high-current MOSFET bank, and closed-loop feedback. Designed in Proteus with firmware developed in Keil µVision.
